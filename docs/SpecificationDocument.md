# Specification Document

## Topic

The topic for this lab is dungeon generation. The practical implementation will be in Unity using C#. 
The purpose of this lab is to get a deeper understanding of the Boyer-Watson algorithm as well Prims algorithm, Wave Function Collapse, A*, and how to implement them.

## Tools & Languages
The algoritm will be implemented in C#, using Unity for graphics.

## Algorithms
- Boyer-Watson
- Prims
- Wave Function Collapse
- A*

All available documentation will be written in english in accordance with international standards.
